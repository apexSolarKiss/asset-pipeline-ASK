# Placeholder Collection Merchandising Constraint Layering Note

This note clarifies current constraint scope within the placeholder `collection / merchandising` context and remains provisional.

## Grouping-Level Constraints

Grouping-level constraints are the constraints that make the set legible as one intentional arrangement rather than as isolated products placed near one another by accident. In the current placeholder collection / merchandising example, these include grouping truth, overall merchandising coherence, and the expectation that the sofa, coffee table, and accent chair read as one governed living-room set.

## Relationship-Level Constraints

Relationship-level constraints govern how items relate to one another inside the set. In the current placeholder collection / merchandising context, adjacency is a spatial relationship constraint rather than a composition preference, and spacing is a governed relationship-bearing constraint rather than merely aesthetic looseness or tightness. These constraints also include relational legibility: each item's role inside the grouped set should remain understandable through spacing, overlap, orientation, and positional emphasis.

## Item-Level Constraints

Item-level constraints preserve product-truth fidelity inside the grouped set. In the current placeholder collection / merchandising context, each item should still remain faithful enough to its own identifying form, finish, and material behavior that grouped coherence does not come at the cost of product truth.

## Where Tension Appears

The main tension in this mode appears where grouped-set coherence and item-level truth pull against one another. A set can still read as calm, coherent, and merchandisable while spacing, overlap, or emphasis choices make one item's role or identifying form insufficiently legible as itself. The current collection negative case makes that pressure explicit: grouped coherence can succeed while one item's product-truth clarity fails.

This is what makes the collection / merchandising burden distinct from both single-item SKU truth and campaign/editorial coherence. The governing challenge is not only whether one item remains faithful or whether one asset family remains directionally coherent, but whether multiple items remain legible as one governed relationship-bearing set without sacrificing item-level truth.
