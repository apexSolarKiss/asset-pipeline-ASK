This index is a reading-order and navigation guide for the repository docs.

It groups existing docs without moving or renaming them.

It is not repo-local doctrine by itself. Linked docs remain authoritative for their subject matter.

It should help a new or returning reader understand what to read first, what is active, what is historical, and what is trigger-based.

Screenshots and generated images are not committed evidence unless a future evidence archive is explicitly created.
