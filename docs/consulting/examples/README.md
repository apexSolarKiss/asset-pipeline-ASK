# Consulting Examples

These examples are compact, abstract consulting case sets used to test the discovery -> diagnosis -> redesign sequence.

## Campaign Example Set

- [`campaign-workflow-discovery-example.md`](campaign-workflow-discovery-example.md): filled discovery example for an abstract campaign workflow
- [`campaign-workflow-diagnosis-example.md`](campaign-workflow-diagnosis-example.md): filled diagnosis example derived from that discovery case
- [`campaign-workflow-redesign-example.md`](campaign-workflow-redesign-example.md): filled redesign example derived from that diagnosis case

These examples are provisional, brand agnostic, and advisory rather than implementation plans.
